  + Polygon size reduction using relative axes

  + Angle-based edge pruning
    + Calculate angles for every three points
    + Remove points with angles smaller than 30 or 40 degrees
    ```
    from math import atan2, degrees

    def angle_between_three_points(self, points):
        x1, y1 = points[0]
        x2, y2 = points[1]
        x3, y3 = points[2]
        deg1 = (360 + degrees(atan2(x1 - x2, y1 - y2))) % 360
        deg2 = (360 + degrees(atan2(x3 - x2, y3 - y2))) % 360
        return deg2 - deg1 if deg1 <= deg2 else 360 - (deg1 - deg2)
    ```
